# WiiTransfer

Bem-vindo ao repositório do projeto **WiiTransfer**.

## Sobre o Projeto

Este projeto é uma aplicação web desenvolvida com tecnologias modernas.

## Tecnologias Usadas

- Next.js
- React
- TypeScript
- Tailwind CSS (se aplicável, verificarei se está no package.json posteriormente, mas por hora deixarei genérico ou verificarei os arquivos)
- Biome (linter/formatter)

## Documentação
Para entender a estrutura de pastas e regras de desenvolvimento, consulte [ARCHITECTURE.md](docs/ARCHITECTURE.md).

## Como Rodar Localmente

1. Clone o repositório:
```bash
git clone https://github.com/smiezi/witransfer.git
```

2. Instale as dependências:
```bash
npm install
# ou
pnpm install
# ou
yarn install
```

3. Rode o servidor de desenvolvimento:
```bash
npm run dev
```

4. Acesse `http://localhost:3000` no seu navegador.
